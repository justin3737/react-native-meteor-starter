# react-native-meteor-starter

## Getting Started

- [Install Meteor](https://www.meteor.com/install)
- [Install React Native](https://facebook.github.io/react-native/docs/getting-started.html) and its dependencies

### Meteor

- `cd meteor`
- `meteor npm install`
- `meteor`

### react-native

- `cd react-native`
- `npm install`
- `npm start`

### ios
- `npm run ios`

### android
- `npm run android`
